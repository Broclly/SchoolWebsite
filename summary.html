Tag Dictionary

  • Table-related tags

Tag Name Description
<table> Creates a table!
<tr> Creates a new row!
<th> Creates a table header, usually used only on the first row!
<td> Adds data, such as words in a data cell!

  • List-related tags

Tag Name Description
<ul> Creates an unordered list!
<ol> Creates an ordered list!
<li> Adds an item to a prexisting list!

  • Link-related tags

Tag name Description
<a href="https://www.example.com"> Initalizes a hyperlink, then requires a universal resource locator, in order to function. Jumps to the selected website after intialization, and required properties.
<a href="example.html"> Initalizes an local link, which then requires a valid .html document to jump to (.html document must be in the computer, if website isn't hosted, or in folder of the website if it is hosted)
<a href="#exampleID"> Initalizes a bookmark jump, which then requires a valid bookmark id which should've been initalized at some point in the website (img, header, text, etc).
<a target="_blank"> An add-on property for any link jump, which forces the browser to make a new tab in order to open the selected link.
<img src="img.example"> Links an image to your website and displays it. Requires having the photo on your computer locally, or requires it to be in the website's folder (if hosting)

  • Text-formatting tags

Tag name Description
<h1> Creates a header for text and website organization. The number signifies the size of which the header's font-size is. It can range from h1 to h7.
<hr> Creates a line on the horizontal axis in the code. Usually used for organizational purposes, or to demonstrate a difference in content in webpages.
<br> Creates a barrier in the text which forces a carriage return, usually used in long forms of text, such as descriptions, paragraphs, biogrpahies, etc.
<p> Allows typing of text, used for any form of text that doesn't declear a header. Highly common in modern websites.
<q> Same as <p> but instead, automatically adds double quotes around the text typed inside the tag. Can be used in combination with <p> as well.
<b> or <strong> Make words bold!
<i> or <ems> Makes words itallic or slanted!
<small> Makes words smaller than the default of <p>
<sub> Makes words subscript!
<sup> Makes words superscript!
<del> Draws a line through words (strikethrough)!
<ins> Adds underline underneath words!
<blockquote> Creates an indent in the words!
<abbr> Adds a dotted line underneath words, which can be used to clarify what certain words mean!
<mark> Highlights words!
  • Critically-important tags
Tag name Description
<html> Specifies that the language being used is html!
<head> Initalizes editing of the tab's properties that page will be opened on!
<title> Edits the name of the tab when a specified window is opened!
<body> Initalizes editing of the webpage!
