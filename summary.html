Tag Dictionary

  • Table-related tags

Tag Name Description
<table> Creates a table!
<tr> Creates a new row!
<th> Creates a table header, usually used only on the first row!
<td> Adds data, such as words in a data cell!

  • List-related tags

Tag Name Description
<ul> Creates an unordered list!
<ol> Creates an ordered list!
<li> Adds an item to a prexisting list!

  • Link-related tags

Tag name Description
<a href="https://www.example.com"> Initalizes a hyperlink, then requires a universal resource locator, in order to function. Jumps to the selected website after intialization, and required properties.
<a href="example.html"> Initalizes an local link, which then requires a valid .html document to jump to (.html document must be in the computer, if website isn't hosted, or in folder of the website if it is hosted)
<a href="#exampleID"> Initalizes a bookmark jump, which then requires a valid bookmark id which should've been initalized at some point in the website (img, header, text, etc).
<a target="_blank"> An add-on property for any link jump, which forces the browser to make a new tab in order to open the selected link.
<img src="img.example"> Links an image to your website and displays it. Requires having the photo on your computer locally, or requires it to be in the website's folder (if hosting)
